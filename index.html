⚙️ Configurar Assinatura

⚠️ Instruções

  1. Preencha os campos acima.
  2. Clique em Gerar Assinatura para visualizar a prévia.
  3. Para usar a assinatura:
    • Outlook Desktop → Arquivo → Opções → Email → Assinaturas… → crie/edite a assinatura e cole a assinatura gerada na área de edição. Compatível com Desktop (Windows/Mac).
    • Outlook Web → Configurações (⚙) → Email → Redigir e responder → Assinatura → cole a assinatura na área de edição.
    • Outlook Mobile → Configurações → Assinatura → cole o conteúdo (algumas versões simplificam a formatação, mas a estrutura permanece).
    • Gmail → Configurações (⚙) → Ver todas as configurações → Geral → Assinatura → crie/edite e cole a assinatura gerada na área de edição. Se preferir usar imagem: clique no ícone de imagem e faça Upload do PNG baixado.
  4. Se quiser salvar, use os botões abaixo da pré-visualização:
    • 📥 Baixar Assinatura (PNG) → salva a assinatura como imagem para colar ou enviar.
    • 💾 Download HTML → gera o código para uso pela equipe de TI ou em outros clientes de e-mail.

Caso surgirem dúvidas, entre em contato com o time de Gestão para mais informações.

✅ Assinatura compatível com Outlook (Desktop, Web e Mobile) e Gmail.
